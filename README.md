# Resume_Matcher
# AI Resume Screening System (ML + NLP)

This project is an NLP-based Resume Screening System that matches resumes with job descriptions using Machine Learning techniques.

## Features
- Resume and Job Description Matching
- TF-IDF Vectorization
- Cosine Similarity Scoring
- Candidate Ranking based on Relevance

## Tech Stack
- Python
- Scikit-learn
- NLP
- React (TypeScript)

This system extracts textual features from resumes and job descriptions and computes similarity scores to rank candidates based on relevance.
